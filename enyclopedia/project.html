О проекте

Цель работы: создать онлайн-энциклопедию со статьями на тематику Новейшей Истории России.

Задачи :

  • Изучить различные источники информации по данной тематике.

  • Представить найденный материал в виде удобного и интуитивно понятного электронного ресурса.

  • Оформить информацию в виде структурированного материала с медиа-вставками для лучшего понимания материала.

  • Реализовать мобильную версию электронного ресурса (кроссплатформенность)

Инструменнты :

  • Сайт написан на языке HTML с добавлением CSS.

  • Для лучшего визуального восприятия был использован CSS-фреймворк Twitter Bootstrap.

  • Был использован редактор кода Microsoft Visual Code.

  • Хостинг предоставлен сервисом Git.Hub.

Актуальность :

Онлайн-энциклопедия может быть использована для личного ознакомления, расширения кругозора, устранения исторической неграмнотности (что очень актуально в наше время), или же как платформа для научной работы студентов и преподавателей. (написание статей, рефератов, исследовательских работ)

Новизна :

Данный ресурс может считаться актуальным, поскольку :

  • он охватывает временные рамки, которые являются актуальными для большей части населения России.

  • в нём содержится структурированная информация по основным события современной истории России, которую обязан знать любой уважающий себя гражданин РФ.

  • энциклопедия содержит биографии некоторых современных политиков и известных людей, что позволяет читателю узнать больше информации о конкретной персоне, чтобы сделать соответствующие выводы.

  • на сайте имеется статическая версия рейтинга статей, без подключения к базам данных, данное решение при дальнейшем развитии позволит отслеживать социальую позицию общества по тем или иным вопросам.

Данный сайт содержит адаптивную вёрстку и полностью поддерживает мобильные устройства для универсального доступа, ведь не все могут позволить себе постоянно использовать комьютерную версию сайта. С помощью данного решения сайтом можно воспользоватья в общественном транспорте, стоя в очерди, находясь в университете.
